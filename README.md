# Evento – Event Management Database Project

Evento is designed for managing events, users, and venues.

This project was created for a Database Systems course to demonstrate full-cycle database design, including ER modeling, normalization, and SQL-based implementation.

# Project Summary

Evento models a real-world event management platform.

The system supports:

Managing users who attend events

Storing event information such as dates, locations, and descriptions

Organizing venues and their capacities

Allowing users to book tickets for events

Running queries to understand attendance, popularity, and operational insights

The focus of the project is on database design accuracy, efficiency, normalization, and relational integrity.

# What This Project Demonstrates

Understanding of Entity–Relationship modeling

Ability to convert requirements into a normalized relational schema

Creation of functional dependencies

Designing meaningful queries and reports for decision-making

Presenting a clean and logically connected database architecture
